\# Anenji 6.2 kW 48 V ESPHome



ESPHome configuration for monitoring and controlling an Anenji 6.2 kW 48 V hybrid inverter over Modbus/RS232.



Based on \[`syssi/esphome-smg-ii`](https://github.com/syssi/esphome-smg-ii) and \[`kodxpl/esphome-anenji`](https://github.com/kodxpl/esphome-anenji). Verify register mappings before writing settings.



Used with Wemos D1 Mini ESP8266 and RS232 to TTL converter
